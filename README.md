# hmthien050209.github.io

My personal blog website

Built with [Astro](https://astro.build), [Vue](https://vuejs.org) and
[TailwindCSS](https://tailwindcss.com)

## How to run?

```shell
# Dev
pnpm dev

# Build for production
pnpm build

# Preview
pnpm preview
```

## Linting and formatting

```shell
# Lint
pnpm lint

# Lint and fix
pnpm lint:fix

# Format
pnpm format

# Format and write
pnpm format:write
```
